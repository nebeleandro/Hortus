Producto

Partiendo de la idea del cliente desarrollamos tres primeras ideas. Luego el cliente decide con cual avanzar, se hacen tres correcciones para llegar al producto ideal. Una vez terminado el diseño, hacemos un prototipo para cerrar esta etapa. Luego se desarrolla la parte de técnica, logrando así poder producirlo de la forma adecuada y posterior venta a la escala deseada.

Espacio

Se proponen tres tipos de mobilario según la línea de diseño que busca el cliente. Una vez elegido el mobilario se avanza en desarrollar la segmentacio del espacio y de ser requerido se desarrollan cambios estrucuturales del espacio.

Animación

Se crean animaciones, videos, posteos o gráficas partiendo de un brandig del cliente o para un producto.

Personaje

Partiendo de la idea del cliente se diseña y modela un personaje.

Presentación

Se diseña imagenes donde se pueda observar el producto correctamente y se crean posteos de este con todos sus atributos.

Modelado

Se modela un producto ya sea para su posterior produccion o para generar contenido con éste.
